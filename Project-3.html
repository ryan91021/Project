Introduction

Reinforcement learning is a type of machine learning where an agent learns to make decisions by performing actions in an environment to maximize cumulative reward. OpenAI Gym is a toolkit for developing and comparing reinforcement learning algorithms. In this project, we will refine the reward function provided by OpenAI Gym using an alternative strategy

Preliminaries

Deep Q-Learning (DQN) is an advanced form of Q-Learning that leverages deep neural networks to approximate the Q-value function, which is used to determine the optimal action to take in a given state.

Exploration: With probability ϵ, the agent selects a random action. This allows the agent to explore the environment ​​and gather more information about the potential rewards of different actions.   

Exploitation: With probability 1−ϵ, the agent selects the action that currently has the highest estimated reward according to the Q-value function. This leverages the knowledge the agent has already acquired to maximize the reward

A fixed-size memory buffer is used to store a limited number of the most recent experiences. When the buffer is full, old experiences are replaced by new ones in a circular manner.

CartPole-v0

The CartPole-v0 environment is a problem that involves a pole attached to a cart that moves along a frictionless track. The objective is to prevent the pole from falling over by applying forces to move the cart either to the left or right. 

next

Refined reward function

x_threshold - |x| calculates how far the cart is from the edge of the allowed range. The closer the cart is to the center, the higher this value. Dividing by x_threshold normalizes this value to a range of 0 to 1. Subtracting 0.8 shifts the reward range to be between -0.8 and 0.2, with higher rewards when the cart is closer to the center.

theta_threshold_radians - |θ| calculates how far the pole is from reaching the maximum allowed tilt. The closer the pole is to vertical, the higher this value. Dividing by theta_threshold_radians normalizes this value to a range of 0 to 1. Subtracting 0.5 shifts the reward range to be between -0.5 and 0.5, with higher rewards when the pole is closer to vertical.
