Introduction

This project focuses on developing a copyright protection scheme for digital images that avoids embedding the watermark directly into the host image. Instead, it encrypts the watermark with the host image using Discrete Wavelet Transformation (DWT) and Singular Value Decomposition (SVD) to extract robust features. These robust features are then X-OR with the watermark to generate the Ownership Share. Additionally, this project will demonstrate resistance to image attacks.
